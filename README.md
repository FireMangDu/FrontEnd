## 🤖 프론트엔드 아키텍처 (Frontend System Architecture)

앱/웹 클라이언트를 중심으로 UI 요청과 실시간 데이터 처리 흐름 다이어그램  
(사용자 입력 → React/TS 기반 UI 처리 → API 호출 → Spring Boot 서버 → MySQL/JPA/MyBatis 데이터 조회 → WebSocket/FCM 실시간 데이터 수신 → UI 업데이트)
